<h2 align="center">Docker入门教程</h2>
</p>

我们把远程服务器上部署好的环境 打包成一个容器，方便我们在未来的某个时刻某个空间一键部署我们的服务，web站点，安装包等等。

优点：
- 减少部署时间
- 方便快捷迁移
- 提升开发效率
- 隔离应用
- 整合服务器
- 调试能力
- 多租户环境

缺点：
- 开发部署问题
- 文件缓存过大
- 64位机器上运行
