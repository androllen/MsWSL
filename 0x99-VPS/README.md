# ShadowsocksR 科学上网原理

SSR从本质上来说是一种加密代理方式，其服务端搭建在国外VPS服务器上，客户端在用户终端设备上。服务器做为中间桥梁，使我们间接访问国外网站。同时由于是加密代理，防火墙无法识别访问内容，访问比较安全。
